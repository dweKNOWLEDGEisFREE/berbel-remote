# berbel-remote
Berbel BFB 6bT Remote Bridge  ESP32-basiert Emulator für Berbel Dunstabzugshauben. Emuliert die BLE-Fernbedienung (inkl. MAC-Spoofing &amp; NimBLE) und bindet die Haube via MQTT Auto-Discovery nativ in Home Assistant ein.  Features: Licht (oben/unten), Lüfter (1-3, Power), Lift-Steuerung &amp; Status-Feedback.  Extras: Web-UI &amp; OTA.
